アタックフィーバー　説明書

■このゲームについて
最終目標は町の周辺を荒らすモンスターのボスを倒すことです
次々と出てくるモンスターから様々な攻撃を用いて戦いましょう


■起動方法
「Game.exe」を実行してください

■基本操作
[Xboxコントローラー]
A				：決定、ジャンプ
B				：ジャンプ
X				：攻撃
X長押し				：回転切り
Y				：必殺技
左スティック、左十字ボタン	：キャラ移動、選択カーソル移動
右スティック、右十字ボタン	：カメラ移動
Start				：オプション移行


※一部のキーの割り当てはメニューの「設定」で変更可能です。

■使用ライブラリ
Dxライブラリ,Effekseer
